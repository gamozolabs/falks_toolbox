# Falk's Toolbox

A collection of single-file, offline-first tools for machinists and makers.

## Core Principles

- **Zero dependencies** — No frameworks, no libraries, no CDNs. Just HTML, CSS, and vanilla JavaScript.
- **Offline-first** — Save any tool to your computer (Ctrl+S) and use it forever. No internet required.
- **No ads, no tracking** — These tools respect your time and privacy.
- **Single-file** — Each tool is one self-contained HTML file. Drop it anywhere and it works.
- **Print-friendly** — Every tool includes a print stylesheet for clean black & white output on paper.

## Tools

### [Tap Drill Calculator](tap_drill_calc.html)

Interactive calculator for determining tap drill sizes based on thread engagement percentage. Features:

- UNC, UNF, and Metric Coarse thread tables
- Adjustable thread engagement (40-85%) with color-coded recommendations
- Drill oversize compensation using data from Machinery's Handbook p.897
- Drill size lookup: Metric, Decimal, Fractional, Letter, and Number drills
- Unified control panel with real-time "nutrition facts" style results
- Click any table row to populate the calculator for detailed info
- Color-coded quality indicators (how close each drill is to ideal)
- Formulas from Machinery's Handbook using correct 1.08253 constant for Unified/ISO threads

## Usage

1. Download the HTML file(s) you need
2. Open in any modern web browser
3. Bookmark or save locally for offline use
4. Print directly from the browser when needed
